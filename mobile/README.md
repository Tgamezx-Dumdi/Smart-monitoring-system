# Smart System Monitor

Smart System Monitor is a full-stack monitoring application designed to visualize system health metrics in real time through a modern mobile dashboard.

The project combines a Flutter mobile application with a Node.js backend to monitor and display system statistics such as CPU usage, memory utilization, system uptime, and device activity. It demonstrates how mobile applications can interact with backend services to provide live operational insights.

## Features

- Real-time system monitoring
- CPU and memory usage tracking
- System uptime monitoring
- Mobile dashboard built with Flutter
- Backend API built with Node.js and Express
- Clean and responsive UI

## Tech Stack

Frontend
- Flutter
- Dart
- Material UI

Backend
- Node.js
- Express.js

## Purpose

This project was built as a learning project to demonstrate full-stack development using Flutter and Node.js while implementing a simple real-time monitoring system.

It also serves as a portfolio project showcasing mobile app development, backend API integration, and system monitoring concepts.
